# Glycemia_log
A simple Python/Tkinter app for tracking daily glucose levels and insulin doses. Designed to help you log your health data easily and visualize weekly summaries.
Features

- Add Todayâ€™s Log: Enter your daily glycemia and insulin readings.
- Weekly Summary: View logs for the week, organized by Morning, Lunch, Dinner, and Bedtime.
- Search by Date: Quickly see the log for any specific day.
- Persistent Storage: All logs are saved in a JSON file (daily_log.json).
- Standalone App: Can be packaged into an executable for easy use without Python.
